# Analysis

This folder contains the statistical analysis scripts and anonymized datasets used in this study.

The detailed descriptions of the experimental design, statistical methods, and results are provided in the corresponding paper.

---

## Folder Structure
analysis/
├── RQ1/
│   └── R scripts for RQ1 analyses
│
├── RQ2/
│   └── R scripts for RQ2 analyses
│
├── data/
│   └── Anonymized datasets used for statistical analyses
│
└── making_heatmap.R
└── R script for generating spatial distribution heatmaps

---

## Description

### `RQ1/`

This folder contains the R scripts used for the analyses corresponding to RQ1.

The scripts include analyses of:

- the overall effect of experimental factors on the illusion of instrument presence
- analyses considering participants' musical experience

---

### `RQ2/`

This folder contains the R scripts used for the analyses corresponding to RQ2.

The scripts include analyses of:

- the relationship between the illusion of instrument presence and aesthetic evaluation
- analyses considering participants' musical experience

---

### `data/`

This folder contains the anonymized datasets used for all statistical analyses.

No personally identifiable information is included.

---

### `making_heatmap.R`

This script generates heatmaps visualizing the perceived spatial distribution of the virtual instrument.

The generated visualizations correspond to the spatial analysis described in the paper.
